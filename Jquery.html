1.What is JQuery?

JQuery is a lightweight "writless, do more" Javascript library.The purpose of JQuery is to make it much easier to use Javascript on your website.

2.How to apply CSS using JQuery,How to add class and Remove class in JQuery, JQuery animation?

TO add a new class,we use JQuery addclass()method.The JQuery toggle class() or remove one or more classes from the selected elements in such a way that if the selected element already has the class.
JQuery animate()method is used to create custom animation.

3.How to create slider with animation?

In JQuery ,we can proved various types of animation using the animate()method. This method can produce simple,to complex animation in the web page.
We apply change to the properties by producing parameter of the method.
Syntax:----
&("selector").animate({params,speed,callback})
.param parameter specifies the css properties to be changed during execution of animate() method.It is a required parameter.
.Speed parameter specifies the speed at which the effect is applied.
.Callback parameter specifies the function to be execution of animate()method.
